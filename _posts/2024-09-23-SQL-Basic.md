# SQL 기초

## 데이터 조회

### SELECT
* 테이블과 컬럼에서 데이터를 조회
* select 컬럼1, 컬럼2(부분 조회) 또는 * (전체 조회) from 테이블

>문자, 날짜_ - '' 이용 ex)'korean' 또는 '2024-10-19'<br>
>별명 설정   - 컬럼 as 별명 또는 컬럼 별명<br>
>특수문자, 한글인 경우 "별명"

### WHERE
* 같지 않다 <>
* IN은 포함 조건 (조건1, 조건2) 사용
* LIKE는 유사한 문자열 조건 

>''와 % 사용 ex) '%자열' 또는 '문%열' 또는 '문자%'

* BETWEEN 조건1 and 조건2 조건 사이에 and를 사용
* and는 동시 충족, or 하나만 충족
* not은 충족하지 않는 조건에 사용

3. DISTINCT
* 중복 없이 컬럼의 데이터를 조회
* select distict 컬럼
